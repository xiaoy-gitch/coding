## 📁 仓库题型分类
仓库精准对标互联网面试高频考点，全覆盖主流算法与数据结构专项，分类清晰、体系完整：

- **数组与字符串**：双指针、滑动窗口、前缀和
- **链表专项**：链表反转、快慢指针、链表合并与分割、环形链表、链表相交等经典题型
- **哈希与集合**：快速查找、数据去重、频次计数、两数/三数/四数之和系列高频题
- **栈与队列**：基础模拟、括号匹配、单调栈、滑动窗口最大值等经典应用题型
- **二叉树专项**：前/中/后/层序遍历（递归+迭代）、树的深度、路径问题、二叉搜索树操作
- **回溯算法**：组合、排列、子集、字符串分割、棋盘回溯等经典模板题
- **动态规划**：基础DP入门、各类背包问题、子序列问题、打家劫舍、区间DP等高频题型
- **贪心算法**：区间覆盖、跳跃游戏、最优选择、局部最优解类经典题型
- **二分查找**：基础二分、边界精细处理、二分答案、极值查找高频应用题
- **高频面试热题**：大厂校招/实习高频考题、经典压轴难题、面试必刷核心题

## 📄 单题完整内容规范
仓库内每一道题目均为 **一站式完整学习资料**，覆盖刷题、理解、复盘、优化全流程：
